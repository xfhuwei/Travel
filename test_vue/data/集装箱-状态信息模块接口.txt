1、获取抵港、海关放行及卸船时间 和进口航次
url：http://${ip}:${port}/gzhg/pier/getUnloadAndBerthTime?billNo=${id}&containerNo=${containerNo}
返回结果：{
      "code":0,
      "msg": "success",
     "data": {
        "releasedTime":"2018-01-11"，//海关放行时间
	    "unloadTime":"2018-01-11",//卸船时间
	    "berthTime":"2018-01-11"，//抵港时间
		"imVoyage":"100xc" //进口航次 //
    
      }
 }
 
 
 
 
2、根据集装箱编号、提单号和进口航次 获取海关舱单详细信息。
url：http://${ip}:${port}/gzhg/customs/getCustomsManifestInfosByContainer?billNo=${id}&containerNo=${containerNo}&importVoyage=${importVoyage}
进口航次在1接口有返回
取舱单申报时间作为界面左下角的显示时间，舱单信息编号为页面左下角显示的单号。
返回结果：{
      "code":0,
      "msg": "success",
     "data": [{
        "xxx":"xxx"，
        "xxx":"xxx"，
    
      }]
 }

 3、根据集装箱编号和进口航次 获取最新报关单号及时间
  url：http://${ip}:${port}/gzhg/customs/getNewestEntry?containerNo=${containerNo}&importVoyage=${importVoyage}

返回结果：{
      "code":0,
      "msg": "success",
     "data": {
        "dDate": "2018-01-11", //时间
        "entryId": "516620181668009760" //报关单号
    
      }
 }
 

 
 
 
4、根据报关单编号(3接口返回)，获取取查验单号及时间

      "msg": "success",
     "data": {
        "dDate": "2018-01-11",//时间
        "examRecId": "516620181668009760" //查验单号
    
      }
